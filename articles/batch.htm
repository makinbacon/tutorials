Code a simple batch program

Last edited: 1st January 2016


You need a Microsoft Windows computer to do this! I am going to show you how to code a simple program that displays a few messages and closes.


Step 1: Open notepad.

Step 2: Enter in @echo off

Step 3: Take a new line. Write the word title then write your name. (example: title John Smith)

Step 4: Take a new line again. Write echo and a greeting (example: echo Hi there!)

Step 5: Take another new line. Type pause

Step 6: Take a new line and write echo This is my first program!

Step 7: Press Save As and title the file Test.bat

Step 8: Change Text Documents (.txt) at the bottom to All Files

Step 9: To run your file, open Command Prompt on your computer, drag the file you made into it and press the enter or return key on your computer
